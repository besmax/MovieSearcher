# MovieSearcher
Приложения для поиска информации о фильмах с IMDB API.

## Используемые технологии и инструменты
- MVVM
- Single Activity
- Navigation Component, Safe Args
- ViewBinding
- Hilt
- Retrofit
- Room
- Kotlin Coroutines
- RecyclerView, DiffUtil
- Glide
- ViewPager2
